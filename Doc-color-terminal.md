# Couleur du print dans un terminal


> forme du texte

- **bold** :
```py
\033[1m
``` 
- __underline__  :
```py
\033[4m
``` 
- blink      :    \033[5m
- reverse    :    \033[7m
- concealed"  :    \033[8m
